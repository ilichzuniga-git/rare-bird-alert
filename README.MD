# Rare Bird Alert (RBA)

Mobile app that surfaces rare bird sightings in Los Angeles and Orange Counties (extensible to more), pulled from eBird and other sources, with push notifications.

## Structure

- `backend/` — Express API + poller (Node)
- `mobile/` — React Native app (Expo)
- `docs/` — Notes, architecture, data source legal posture
